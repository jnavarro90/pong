Descripción detallada

Clase que representa el socket remoto de la conexión.

Se usará para aceptar paquetes y conexiones entrantes al programa.

CHostSocket::CHostSocket ( Uint16  port)

Constructor principal.

Parámetros
portPuerto para establecer la conexión.

Se encarga de iniciar el objeto del socket a partir de un puerto dado. Se generará una estructura de tipo CIpAddress a partir del puerto. Posteriormente, se intentará crear el socket. En caso de no poder crear el socket, SDL_net generará un error.

bool CHostSocket::Accept ( CClientSocket &  the_client_socket)

Acepta conexiones entrantes.

Devuelve
En caso de no poder aceptar ninguna conexión, devolverá false. Si se puede aceptar una conexión entrante, devolverá true.

Se intentará entablar una conexión con un cliente remoto por medio de la función SDLNet_TCP_Accept(). Si todo va bien, se entablará la conexión.
