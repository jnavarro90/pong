Descripción detallada

Clase de estancia para representar el modo de juego multijugador-online desde el servidor.

Básicamente, encapsula algunos objetos (2 pads, 1 pelota y 1 marcador) para trabajar con ellos de una forma más sencilla. El cservidor, primero entrará en un menu desde el que debe esperar la conexión de algún cliente (con el puerto NET_MJ_PORT). Una vez realizada la conexión, el usuario enviará la configuración de la partida, ubicada en COpciones. Posteriormente, el bucle de la partida en el servidor es el siguiente:

Mientras(ejecutando)
  Gestionar datos de entrada
  Recibir datos de entradadel cliente
  Procesar datos de entrada
  Enviar datos procesados al cliente
  Mostrar por pantalla
FinMientras

Por defecto, el pad del servidor es el izquierdo.

Para más información sobre el funcionamiento de las estancias, vea las definiciones de los métodos de esta clase y las clases CInstance, CEngine, CMenu y CBoton.

int CInstance_MJ_Online_Server::OnExecute ( )
virtual

Prototipo de función de ejecución.

Devuelve
Devuelve la próxima estancia a ser ejecutada por el bucle principal ubicado en CEngine::OnExecute()

Ejecuta el bucle principal de la estancia que se estructura de la forma siguiente:

Init() // Iniciar
while(ejecutando) // Bucle principal
{
OnEvent() // Gestionar eventos
OnLoop() // Gestionar datos
OnRender() // Mostrar por pantalla
}
Close() // Cerrar
return estancia_siguiente;
