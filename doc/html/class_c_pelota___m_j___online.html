Descripción detallada

Clase para representar la pelota del modo de juego multijugador-online desde el servidor.

Los objetos de esta clase funcionan de forma parecida a los de la clase CPelota_MJ_Local, salvo que en esta, al mover la pelota, si se debe reproducir algún sonido, se deberá notificar al usuario enviándole una notificación. Véase CNetwork_Data_Server::fs
