Descripción detallada

Clase para dibujar marcos.

Esta clase se encarga de crear una superficie en forma de "marco", con un color de fondo, un color de marco y un determinado grosor. La superficie se genera en la llamada al constructor, y se libera cuando se destruye el objeto.
