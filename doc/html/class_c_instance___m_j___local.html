Descripción detallada

Clase de estancia para representar el modo de juego multijugador-local.

Básicamente, encapsula algunos objetos (2 pads, 1 pelota y 1 marcador) para trabajar con ellos de una forma más sencilla. El modo consiste en un sistema de pong multijugador básico. Por el momento el jugador 1 juega con las teclas W y S, mientras que el jugador 2 usa las teclas ARRIBA y ABAJO. Para más información sobre el funcionamiento de las estancias, vea las definiciones de los métodos de esta clase y las clases CInstance, CEngine, CMenu y CBoton.

int CInstance_MJ_Local::OnExecute ( )
virtual

Prototipo de función de ejecución.

Devuelve
Devuelve la próxima estancia a ser ejecutada por el bucle principal ubicado en CEngine::OnExecute()

Ejecuta el bucle principal de la estancia que se estructura de la forma siguiente:

Init() // Iniciar
while(ejecutando) // Bucle principal
{
OnEvent() // Gestionar eventos
OnLoop() // Gestionar datos
OnRender() // Mostrar por pantalla
}
Close() // Cerrar
return estancia_siguiente;
