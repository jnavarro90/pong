Descripción detallada

Clase para mostrar avisos (texto) por pantalla.

Esta clase se utiliza para mostrar en la superficie pantalla un texto deseado almacenado en la variable str. Se dibujará con el color guardado en color1 y con la fuente apuntada por ttf_fuente.
