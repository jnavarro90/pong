Descripción detallada

Clase para un sistema de partículas muy simple.

Se utilizará para dejar un rastro de un objeto en movimiento. No utilizado actualmente en el proyecto. El elemento que quiera usar el sistema de partículas, deberá tener un vector de CParticula y actualizar sus posiciones de forma conveniente, en función del movimiento deseado.
