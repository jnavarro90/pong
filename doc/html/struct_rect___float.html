Descripción detallada

Estructura similar a SDL_Rect.

struct Rect_Float
{
float x,y;
int w, h;
}

En vez de usar enteros para las coordenadas, usamos números en punto flotante. A la hora de representarlos por pantalla, se toman los valores enteros.
