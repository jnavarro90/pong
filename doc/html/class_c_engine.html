int CEngine::OnExecute ( )
virtual

Función de ejecucion principal.

La función se encarga de ejecutar el bucle principal del juego, desde el cual se cargarán las diversas estancias. La estructura es bastante sencilla:

int CEngine::OnExecute()
{
if(!Init())
return -1;
int menu = I_MENU_MAIN;
while(running)
{
if(menu == I_SALIDA)
{
running = false;
}
else
{
menu = instance[menu]->OnExecute();
}
}
Close();
return menu;
}

Ademas, las declaraciones del contenido del array de punteros instance se hacen en esta propia función.
