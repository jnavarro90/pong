Descripción detallada

Clase para mensajes de red de N bytes.

Se utiliza como contenedor para mandar datos en red. Sirve para mandar paquetes codificados en un conjunto de caracteres que simbolizan una secuencia de bytes. La cantidad de bytes no debe sobrepasar los 256 bytes. Vea el archivo CNet.h para mayor información de todas las clases de red.

privatevirtual

Indicador de cuantos caracteres hay que cargar.

Devuelve
Devuelve el número de caracteres a cargar.

En caso de que el mensaje está vacío (EMPTY), se ha de cargar completamente, con buffer_size caracteres. En caso contrario, no se puede cargar ningún carácter (FULL).
