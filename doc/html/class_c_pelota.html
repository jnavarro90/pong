Descripción detallada

Clase abstracta para definir la pelota de cualquier modo de juego.

Funciona como clase abstracta para definir las diversas clases pertenecientes a varias estancias, por lo general, una única pelota por estancia. La velocidad está definida en COpciones::PELOTA_VEL, el tamaño en COpciones::PELOTA_ALTO y el color en COpciones::PELOTA_COLOR
