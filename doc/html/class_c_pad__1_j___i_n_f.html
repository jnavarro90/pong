Descripción detallada

Clase general para definir el pad de 1 jugador-infinito.

Minimo de velocidad si se decrementa. El funcionamiento de esta clase es muy simple. Puede ver los métodos de la clase en sí y de la clase base CPad para entender el correcto funcionamiento de esta.

void CPad_1J_INF::eventuar ( )

Gestionar eventos.

Si se pulsa la tecla para subir el pad, se restará a yVel el valor PAD_1J_INF_VELOCIDAD. Si se suelta, se sumará dicho valor. En caso de pulsar la tecla para bajar, se sumará si se aprieta la tecla y se restará si se suelta. Para entender las direcciones, vea el siguiente diagrama de representación de ejes en SDL:
