Descripción detallada

Clase base de mensaje de red.

Se utiliza como contenedor para mandar datos en red. Por defecto, sirve para cargar y descargar cadenas de caracteres. Vea el archivo CNet.h para mayor información de todas las clases de red.

Indicador de cuantos caracteres hay que cargar.

Devuelve
Devuelve el número de caracteres a cargar.

En caso de que el mensaje está vacío (EMPTY), se ha de cargar completamente, con 255 caracteres. En caso contrario, no se puede cargar ningún carácter (FULL).
