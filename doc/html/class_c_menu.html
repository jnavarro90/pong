Descripción detallada

Clase para almacenar conjuntos de botones.

Se encargará de gestionar el uso de un cierto número de botones como si se tratase de un botón. Se verificará, para cada botón, que el ratón esté encima de algún botón y que se haga click izquierdo encima de alguno.

Documentación de las funciones miembro

int CMenu::eventuar ( )

Gestionar eventos.

Devuelve
Devuelve el número del botón pulsado, desde 1 hasta nBotones. Si no se ha pulsado ninguno, devolverá 0.

Si el usuario a hecho click con el mouse, se comprobarán todos los marcos de los botones hasta encontrar alguno que cuadre con la posición del ratón. Si no cuadra con ninguno, se supone que el usuario no ha pulsado ningún botón.
