Descripción detallada

Estructura de datos a mandar desde el cliente al servidor.

Desde el cliente se mandarán sólo los eventos del usuario (si mueve hacia arriba o hacia abajo) y los ticks del temporizador de la estancia del cliente, que posteriormente se comprobarán con los del servidor para realizar ajustes.
