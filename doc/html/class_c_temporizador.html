Descripción detallada

Clase para medir tiempos en milisegundos.

Esta clase se utiliza para medir tiempos de forma sencilla. Es usado principalmente como medido para limitar los fotogramas por segundo, o para contar el timeout en clases relacionadas con la modalidad de red.

int CTemporizador::getTicks ( )

Obtener el número de milisegundos desde que empezó el contador.

Calcula el tiempo en milisegundos que el contador lleva corriendo.

Devuelve
Si el temporizador no ha empezado, devuelve 0. Si hay empezado y está pausado, devuelve el último instante cuando se pausó. Si no está pausado, devuelve el nº de milisegundos que lleva corriendo el temporizador (SDL_GetTicks() - empezadoTicks). Para pasar a segundos, basta con dividir el valor entre 1000.f
