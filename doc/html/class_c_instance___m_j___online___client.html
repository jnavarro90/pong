Descripción detallada

Clase de estancia para representar el modo de juego multijugador-online desde el cliente.

Básicamente, encapsula algunos objetos (1 pad, 1 pad "zombi", 1 pelota "zombi" y 1 marcador "zombi") para trabajar con ellos de una forma más sencilla. El cliente, primero entrará en un menu desde el que debe conectar a la dirección ip del servidor (con el puerto NET_MJ_PORT). Una vez realizada la conexión, el usuario recibirá la configuración de la partida ubicada en el servidor en COpciones, y hará las operaciones oportunas. Posteriormente, el bucle de la partida en el cliente es el siguiente:

Mientras(ejecutando)
  Gestionar datos de entrada
  Enviar datos de entrada al servidor
  Recibir datos procesados del servidor
  Mostrar por pantalla
  Si hay que hacer salto de frames
    Saltar frame
  Si no
    Esperar frame
FinMientras

Por defecto, el pad del cliente es el derecho.

Para más información sobre el funcionamiento de las estancias, vea las definiciones de los métodos de esta clase y las clases CInstance, CEngine, CMenu y CBoton.

int CInstance_MJ_Online_Client::OnExecute ( )
virtual

Prototipo de función de ejecución.

Devuelve
Devuelve la próxima estancia a ser ejecutada por el bucle principal ubicado en CEngine::OnExecute()

Ejecuta el bucle principal de la estancia que se estructura de la forma siguiente:

Init() // Iniciar
while(ejecutando) // Bucle principal
{
OnEvent() // Gestionar eventos
OnLoop() // Gestionar datos
OnRender() // Mostrar por pantalla
}
Close() // Cerrar
return estancia_siguiente;
