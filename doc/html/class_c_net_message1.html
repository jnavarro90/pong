int CNetMessage1::NumToLoad ( )
privatevirtual

Indicador de cuantos caracteres hay que cargar.

Devuelve
Devuelve el número de caracteres a cargar.

En caso de que el mensaje está vacío (EMPTY), se ha de cargar completamente, con 255 caracteres. En caso contrario, no se puede cargar ningún carácter (FULL).
