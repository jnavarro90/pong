Descripción detallada

Clase base (abstracta) para derivar y declarar las diversas estancias del juego.

Su unico propósito es dar un formato equivalente para todas las estancias, aunque no todas las funciones sea iguales.

int CInstance::OnExecute ( )
virtual

Prototipo de función de ejecución.

Devuelve
Devuelve la próxima estancia a ser ejecutada por el bucle principal ubicado en CEngine::OnExecute()

Ejecuta el bucle principal de la estancia que se estructura de la forma siguiente:

Init()
while(ejecutando)
{
OnEvent()
OnLoop()
OnRender()
}
Close()
return estancia_siguiente;
